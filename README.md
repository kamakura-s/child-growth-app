# すくすくノート（子どもの成長記録アプリ）

ローカル（HTML / CSS / JavaScript）だけで動く、子どもの成長記録MVPです。  
データはブラウザの `localStorage` に保存されます。

## 主な機能

- 子どもプロフィール登録（名前・生年月日・メモ）
- 身長 / 体重の記録（日時・メモ）
- 写真付き日記メモ
- 記録一覧表示（成長記録 / 日記）
- 簡易グラフ（Canvas: 身長 / 体重推移）
- 個別削除・全データ削除

## ファイル構成

- `index.html` : UI
- `style.css` : デザイン
- `app.js` : 保存・一覧描画・グラフ・バリデーション

## 使い方

1. `index.html` をブラウザで開く
2. プロフィールを保存
3. 成長記録や日記を追加
4. 一覧とグラフで確認

## 注意

- ブラウザを変えるとデータは共有されません
- localStorage容量を超えると画像保存に失敗する場合があります
- MVPのため、認証・バックアップ・同期は未実装です
